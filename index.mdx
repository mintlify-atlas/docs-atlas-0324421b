The AWS Containers Retail Store Sample is an intentionally over-engineered retail application designed to demonstrate container orchestration patterns, microservices architecture, and AWS-native deployment strategies. It provides a fully functional storefront with product catalog, shopping cart, and checkout — all wired together across five independently deployable services.

## Get started

<Steps>
  <Step title="Choose a deployment method">
    Pick Docker for the fastest local start, Docker Compose for the full stack, Kubernetes for cluster testing, or Terraform for a production-like AWS deployment.
  </Step>
  <Step title="Run the application">
    Follow the quickstart for your chosen platform. The UI is available at `http://localhost:8888` for local deployments.
  </Step>
  <Step title="Explore the features">
    Browse the storefront, inspect the `/info` topology page, enable the AI chatbot, or run the load generator against the live stack.
  </Step>
  <Step title="Customize and experiment">
    Change themes, swap persistence backends, enable Istio or OpenTelemetry, and use chaos endpoints to test resilience.
  </Step>
</Steps>

<Note>
  This application is intended for educational and demonstration purposes. It is not designed for production use.
</Note>
